## Installation

1. Copy .env.example, rename IT, and set the correct ACCESS_TOKEN.
2. Create Postgresql DB, and set the correct DATABASE_URL in .env
3. Run `npm i`
4. Run `npm run prisma:db:push`


## Usage


```bash
node src/index.js
```
